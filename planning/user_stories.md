# User Stories

## Outline User Roles

* **Group Admin**: A user who creates and manages a movie group and its members.
* **Group Member**: A user who has joined a group to participate in suggestions and voting.
* **General User**: An authenticated user viewing their personal dashboard of joined groups.

## Draft User Stories

1. As a Group Admin, I want to create a new group so that I can organize a specific movie night for my roommates.
2. As a Group Admin, I want to delete a member from the group so that the group remains private to our current circle.
3. As a Group Member, I want to suggest a specific movie so that my friends know what I am interested in watching.
4. As a Group Member, I want to rate a movie from 1 to 5 stars so that I can express my level of excitement for that film.
5. As a Group Member, I want to see the voting results update instantly when my friends vote, so that we can discuss the leading movies in real-time.
6. As a Group Member, I want to see the Top 3 liked movies so that I can quickly see what the group's current favorites are.
7. As a General User, I want to see a list of all the groups I belong to so that I can easily navigate between different friend circles.
8. As a Group Member, I want to edit a movie suggestion I made in case I misspelled the title or want to update the description.
9. As a Group Member, I want to delete a suggestion I added if I realize the group has already seen that movie.
10. As a Group Member, I want to sort the movie list by "Highest Rated" so that I can see the most popular choices at the top.
